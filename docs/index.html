# alt:V Multiplayer
Это альтернативная многопользовательская среда для GTAV, которая позволяет разработчикам и игрокам играть на пользовательских серверах. Эти серверы могут использовать такие языки, как C# и JavaScript. Здесь вы узнаете, как работать с alt:V на JavaScript, как правильно настроить проект и создать свой собственный игровой режим.
